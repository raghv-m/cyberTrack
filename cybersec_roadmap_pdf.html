PHASE 0 ‚Äî FOUNDATION

Now ‚Üí January 2026 (3-4 months)

Goal: Build Strong Cybersecurity Fundamentals

Become job-ready for SOC Tier 1 positions with hands-on experience and a portfolio that proves your abilities.

Daily Schedule (2-3 hours minimum)

  • Complete Udemy cybersecurity course (4-6 weeks maximum)
  • TryHackMe SOC Level 1 learning path (primary focus)
  • Hands-on labs with real security tools
  • Build GitHub portfolio with writeups

Core Knowledge Areas

Networking Fundamentals

  • TCP/IP protocol suite and packet structure
  • OSI model (all 7 layers) and real-world application
  • DNS resolution process and common attacks
  • HTTP/HTTPS differences and TLS basics
  • Common ports (22, 80, 443, 3389, 445, 53) and their services
  • Packet flow analysis and network troubleshooting

Operating Systems

  • Windows: Event Logs (Event Viewer), focusing on Security logs
  • Event ID 4624 (successful logon), 4625 (failed logon), 4688 (process creation)
  • Linux: Log locations (/var/log/auth.log, /var/log/syslog, /var/log/secure)
  • Command line basics: grep, tail, cat, ps, netstat
  • File permissions and user management

Security Concepts

  • CIA Triad (Confidentiality, Integrity, Availability)
  • Identity and Access Management (IAM)
  • Principle of least privilege
  • Authentication vs authorization vs accounting
  • Common attack vectors (phishing, malware, brute force)

Tools You MUST Master

Wireshark
Packet capture and analysis
Nmap
Port scanning and enumeration
tcpdump
CLI packet capture
Splunk
SIEM basics and log searching
Windows Event Viewer
Log analysis
Linux CLI
Log parsing and investigation

Hands-On Lab Requirements

  • TryHackMe SOC Level 1: Complete all rooms focusing on alert triage, phishing analysis, and brute-force detection
  • Splunk Labs: Free trial or Splunk fundamentals training (free)
  • Blue Team Labs Online: Practice incident investigations
  • Wireshark Practice: Analyze PCAP files from malware-traffic-analysis.net

Portfolio Output by January 2026 (Non-Negotiable)

Create a GitHub repository containing:

  • 3 complete incident investigation writeups (document your process, findings, and recommendations)
  • 1 detailed phishing email analysis (headers, URLs, attachments, IOCs)
  • 1 brute-force attack investigation (log analysis, timeline, mitigation)
  • Screenshots and explanations of your methodology

Also required: Optimized LinkedIn profile highlighting SOC skills and lab experience

February ‚Üí May 2026 (Job search and first role)

Goal: Get Hired as SOC Tier 1 Analyst

Secure your first cybersecurity position and begin building professional experience.

Job Application Strategy

Start applying in February 2026

Target Job Titles

Skills You Must Demonstrate in Interviews

Skill What They'll Ask
Alert Triage Walk through how you'd investigate a suspicious login alert
False Positive vs True Positive Explain indicators that differentiate legitimate vs malicious activity
Phishing Analysis Describe your process for analyzing a suspicious email
Incident Lifecycle Explain steps: identification, containment, escalation, documentation
Tool Knowledge Discuss experience with SIEM, Wireshark, or any security tools

Core Tier 1 Responsibilities

  • Monitor security alerts from SIEM and other detection systems
  • Perform initial triage on security incidents
  • Analyze phishing emails and malicious attachments
  • Document incidents clearly with timelines and evidence
  • Escalate complex incidents to Tier 2 analysts
  • Update tickets and maintain accurate records
  • Follow security playbooks and SOPs
Expected Salary (Canada): $55,000 - $70,000 CAD
Pro Tip: Night/weekend shifts often lead to faster hiring
Important Advice: Take the first reasonable offer. Don't negotiate yourself out of your first role. Experience is more valuable than an extra $5k at this stage.

Interview Preparation Checklist

  • Prepare 2-3 stories from your lab work using STAR method
  • Know your GitHub projects inside and out
  • Practice explaining technical concepts simply
  • Research the company's security posture and recent incidents
  • Prepare questions about their SOC operations and tools
  • Review common SOC interview questions on Glassdoor

PHASE 2 ‚Äî TIER 1 TO TIER 2 ADVANCEMENT

May ‚Üí December 2026 (7-8 months of growth)

CHOOSING YOUR SPECIALIZATION

Path 1: Cloud Security Engineer

Best for: Big Tech (Amazon, Google, Meta), SaaS companies, modern enterprises

Skills Required:

  • Deep knowledge of AWS/Azure/GCP security services
  • IAM policies, cloud logging, threat detection
  • Infrastructure as Code (Terraform, CloudFormation)
  • Container security (Kubernetes, Docker)

Certifications: AWS Certified Security - Specialty, Azure Security Engineer (AZ-500)

Path 2: Detection Engineer

Best for: Big Tech, MSSPs, companies with large security teams

Skills Required:

  • Expert-level SIEM query writing (SPL, KQL, SQL)
  • Detection rule development (Sigma, YARA)
  • Python for automation and tool integration
  • MITRE ATT&CK framework mastery

Certifications: Splunk Enterprise Security Certified Admin, Blue Team Level 2 (BTL2)

Path 3: DFIR Specialist

Best for: Government, finance, incident response consulting

Skills Required:

  • Memory forensics (Volatility, Rekall)
  • Disk forensics (Autopsy, EnCase, FTK)
  • Malware analysis basics
  • Evidence handling and legal procedures

Certifications: GCFE, GCIH, GCIA, or CHFI

Decision Framework:
  • Want to work at FAANG/Big Tech? ‚Üí Cloud Security or Detection Engineering
  • Want stability, pension, and work-life balance? ‚Üí Government DFIR
  • Want high salary and fast career growth? ‚Üí Detection Engineering
  • Want to lead major incident responses? ‚Üí DFIR Specialist

CRITICAL SUCCESS FACTORS

What Will Make or Break Your Career

‚úÖ Do These Things

  • Build in public: Document everything on GitHub and LinkedIn
  • Network intentionally: Connect with security professionals, attend local meetups
  • Switch companies strategically: 2-3 years per role is normal and healthy
  • Focus on depth: Master a few tools deeply rather than knowing many superficially
  • Automate everything: Show you can save time and scale operations
  • Ask for stretch projects: Volunteer for the hard problems others avoid

‚ùå Avoid These Mistakes

  • Certificate collecting: Don't delay your career waiting for the "perfect" certification
  • Staying too long: If promotion doesn't happen after 12-18 months, leave
  • Tutorial hell: Watching courses without building projects won't get you hired
  • Neglecting soft skills: Communication is as important as technical ability
  • Ignoring job market signals: Apply to 20+ positions, track what gets responses

The Truth About SOC Careers

  • SOC is a launchpad, not the destination. Use it to build experience, then specialize.
  • Big Tech hires engineers, not alert-clickers. Focus on automation and building.
  • Government hires trusted specialists. Clearances and compliance matter.
  • Switching companies accelerates salary growth. Don't expect 30%+ raises from internal promotions.
  • Depth beats certificates every time. Hiring managers value real experience over paper credentials.
